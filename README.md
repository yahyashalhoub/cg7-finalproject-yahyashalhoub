🎮 Coin Runner 3D
📖 فكرة اللعبة
لعبة Coin Runner 3D هي لعبة منصات ثلاثية الأبعاد حيث يتحكم اللاعب بشخصية ماريو ذات القبعة الحمراء. الهدف هو القفز فوق المنصات البنية المرتفعة وجمع 5 عملات ذهبية للفوز. يجب على اللاعب تجنب الأعداء الخضر (السلاحف) التي تتحرك ذهاباً وإياباً، ويمكنه القفز فوقهم لهزيمتهم. الوقت محدود بـ 120 ثانية، وأي سقوط من المنصات يؤدي إلى الخسارة الفورية.

🕹️ عناصر التحكم
المفتاح	الوظيفة
W / ↑	التحرك للأمام
S / ↓	التحرك للخلف
A / ←	التحرك لليسار
D / →	التحرك لليمين
SPACE	القفز للوصول إلى المنصات الأعلى
R	إعادة تشغيل اللعبة
ESC	الخروج من اللعبة
✨ الميزات البصرية
الميزة	الوصف
إضاءة Phong	نظام إضاءة واقعي مع ضوء موضعي يضيء المشهد من الأعلى
شخصية ثلاثية الأبعاد	ماريو مكون من قبعة حمراء، رأس بلون البشرة، جسم أزرق، وأرجل تتحرك أثناء المشي
أعداء متحركة	سلاحف خضراء بعيون بيضاء تتحرك يميناً ويساراً في دوريات
عملات دوارة	عملات ذهبية تدور حول نفسها وتتحرك لأعلى وأسفل بتأثير طفو جذاب
منصات ملونة	منصات بنية بدرجات مختلفة للتمييز بين المستويات
سماء زرقاء	خلفية بلون سماوي مريح للعين
HUD ملون	واجهة عرض في الطرفية بألوان مميزة (أخضر للفوز، أحمر للخسارة، أزرق للعب)
🏆 شروط الفوز والخسارة
الحالة	الشرط
🏆 فوز	جمع 5 عملات ذهبية
💀 خسارة	نفاد الوقت (120 ثانية)
💀 خسارة	السقوط من المنصات إلى الأسفل
💀 خسارة	الاصطدام بعدو من الجانب أو الأسفل
ملاحظة: يمكن للاعب القفز فوق العدو لهزيمته بأمان.

🔧 التقنيات المستخدمة
التقنية	الوصف
OpenGL 3.3 Core Profile	استخدام خط أنابيب الرسم الحديث بدون مكتبات قديمة
GLFW	إدارة النافذة واستقبال المدخلات من لوحة المفاتيح
GLAD	تحميل مؤشرات دوال OpenGL الحديثة
GLM	مكتبة رياضية للحسابات ثلاثية الأبعاد (مصفوفات، متجهات)
VAO / VBO / EBO	تخزين وإدارة بيانات الرؤوس في كرت الشاشة
AABB Collision	خوارزمية تصادم الصناديق المحاذية للمحاور
Phong Shading	نموذج إضاءة في Shader يحسب الإضاءة المحيطة والمنتشرة
Delta Time	فصل منطق اللعبة عن سرعة الإطارات لحركة سلسة
State Machine	إدارة حالات اللعبة (لعب، فوز، خسارة)
